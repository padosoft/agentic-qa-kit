# @aqa/pack-core

Baseline pack — installed in every project. Provides:

- `templates/testing.md.tpl` — starter `.aqa/testing.md` skeleton.
- `templates/risk-map.yaml.tpl` — starter `.aqa/risk-map.yaml`.
- `risks/baseline.yaml` — two universal seed risks (uncaught exceptions, input validation).

`applies_when: {}` — this pack matches every project.
